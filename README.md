# A World of Warcraft bemutatása.

### A World of Warcraft egy 2004-ben megjelent nagyon sok szereplős online szerepjáték. A nagy sikerű Warcraft sorozat játék folytatása.

A játék az MMORPG stílust alkalmazza, ami annyit jelent, hogy online internet kapcsolatot igényel, ezáltal több százan vagy ezren is játszhatják egy időben a játékot. Rendszeres előfizetést igényel.
